# TP_CLUB – Sistema de Gestión del Club Deportivo

Proyecto académico desarrollado de forma individual en **C#**, orientado a la gestión de un club deportivo.

El sistema permite administrar socios, niños, entrenadores, deportes e inscripciones, aplicando conceptos de **Programación Orientada a Objetos (POO)** y lógica de negocio.

## Tecnologías utilizadas
-C#
-.NET
-Programación Orientada a Objetos
-Colecciones
-Manejo de excepciones
-Configuración mediante App.config

## Funcionalidades principales
- Alta, baja y modificación de socios
- Gestión de niños y entrenadores
- Administración de deportes
- Inscripción de socios a actividades deportivas
- Validaciones básicas y control de errores

## Estructura del proyecto
- Clases organizadas por entidades del dominio (persona, socio, niño, entrenador, deporte, club)
- Archivo de configuración del proyecto
- Diagrama UML para el modelado del sistema

## Estado del proyecto
Proyecto funcional, con posibilidades de mejora y refactorización para optimizar la estructura y el diseño del código.
